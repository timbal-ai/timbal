## Usage

```python
from timbal import Agent

agent = Agent(
    name="kimi",
    model="moonshot/kimi-k3",
    model_params={"reasoning_effort": "max"},
    tools=[],
)

result = await agent.collect(prompt="Explain Kimi K3 in one sentence.")
print(result.output.collect_text())
```
